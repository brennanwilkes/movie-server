Movie Night

Brand studies · v5 · four themes, one app · July 2026

New in v5: the Top 100 pantheon expands to ranks 1–10 and goes bespoke — billing blocks (director + cast in poster-credit type), genre/franchise motifs (venetian blinds, map routes, gunbarrel dots), composition variety, tiers re-cut to 1–10 / 11–50 / 51–100. From v4: toolbars now show the real center nav (Home / Search / Top 100 / Watchlist, per MainToolbar) with a themed active state · a new Top 100 tab designs the showcase page that replaces the text-list playlist view — full-bleed pantheon for the top ranks, gallery, then ledger, with automatic era treatments · the Web tab converges on the Fire Stick: pruned sidebar + the same showcase. From v3: home screens now render over live blurred artwork, exactly as the app does while browsing (AppBackground: blur + ~58% dim). Each theme claims the background only through a hue-biased dim wash — never a stronger one, so no artwork brightness is lost vs today. From v2: taglines removed everywhere (the wordmark stands alone) · the hero spotlight row is now mocked accurately (backdrop, tinted scrim, the movie's own clearlogo, themed Play button) · poster cards use the real app format — busy one-sheet art, the actual #7 rank-pill geometry with per-theme stroke & font, titles + runtime below in the theme face · 🎲 Spin simulates the launch roulette. Content is identical across themes; only the chrome changes.

Top 100 — the showcase page — shown in Canyon; chrome follows the active theme

Replaces the text-list playlist page with descending ceremony: ranks 1–10 get full-bleed pantheon panels, each made bespoke by the film's own metadata — billing block (director + top-billed cast in compressed poster-credit type), a genre or franchise motif, era treatment from the release year, and composition variety so ten big rows never repeat. 11–50 pair up as gallery cards; 51–100 become a fast branded ledger. All of it assembled from TMDB/Jellyfin assets and fields — no invented text.

Billing block

Director + top-billed cast set in compressed poster-credit type (scaled condensed caps, wide tracking) — the classic one-sheet billing look, filled entirely from the item's People field.

Genre & franchise motifs

Vector overlays keyed off real fields: noir → venetian-blind shadow; adventure → map contours + dashed route; caper → deco double frame; TMDB collection id → franchise mark (Bond gets the gunbarrel dots). A small motif table maps genre/collection → overlay; unknown genres get none.

Composition variety

Layout is chosen deterministically per film (motif kind + backdrop aspect + rank parity): poster inset, motif corner, logo scale. Ten pantheon rows, no two identical, and stable across visits — not random.

Era treatment (automatic)

From release year: pre-~1955 silver desaturation + silver hairline; 1955–75 technicolor lift + warm frame; 1976–99 amber; 2000+ neutral. Combines with the genre motif.

Focus interaction

On TV, the whole panel is the focus target (Play on click). Stretch idea: focusing a panel whose director appears elsewhere in the 100 subtly glows those ledger rows — the list reveals its own threads. Cheap to compute, zero extra assets.

Open call: taglines

TMDB carries each film's real theatrical tagline ("They said it couldn't be done."). It's authentic studio copy, not generated — but it is added text. In or out, your call; mocked out for now.

Fire Stick reality

Lazy column; backdrops half-res (same trick as AppBackground); motifs are cheap draws; ≤3 full-bleed images resident. Ledger rows are plain views. Must scroll smoothly on the old stick or tiers get shallower.

Header garbage: deleted

No genre list, no total runtime, no item-count banner. The page opens directly on №1.

Theme dialect & Watchlist

Numerals/frames follow the active theme (condensed-amber offset for Matinee, white-on-vermilion discs for Reel One, gold + deco rules for Marquee). Watchlist reuses the component minus rank ceremony: gallery + ledger only.

Web client — converging on the Fire Stick, Canyon-fixed

Custom CSS carries accent, ground, fonts, and the wordmark; the JavaScript Injector (already in place for playlist flair) builds the same Top 100 showcase from the API. The sidebar is pruned to what the family actually uses — no Dashboard, no admin, no syncplay clutter; everything operational lives in IaC anyway.

Sidebar pruning

CSS display:none on Dashboard/admin/metadata links plus the header cast; keep Home, libraries, Top 100, Watchlist. Admin needs are met by IaC — if something must be clicked, the dashboard is still reachable by URL.

Convergence

Same palette, same fonts, same wordmark, same showcase page, same pruned nav set as the Fire Stick toolbar (Home / Movies / Shows / Top 100 / Watchlist). Web stays Canyon while the TV rolls the roulette.

Showcase mechanism

The JS injector detects the Top 100 playlist route, hides the stock list + header junk, and renders the tiers from the same API data (images by item id). Fragile-ish across Jellyfin upgrades — accepted, it's already the pattern used for playlist flair.

Wordmark asset reuse

The outlined-SVG wordmark built for the TV app doubles as the login logo (Dashboard → Branding) — one master asset, two clients.
